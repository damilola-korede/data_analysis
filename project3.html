Apartment Initial Pricing
Analysis & Prediction

Type
Regression · EDA
Model
Ridge Regression Pipeline
Libraries
Pandas · Matplotlib · Seaborn · Plotly · Sklearn · ipywidgets
01 — Overview
Objective

This project aimed to analyze the initial costs of apartments across different New York boroughs, identify pricing trends, and predict future values using regression models.

The dataset was sourced from Kaggle and contained apartment initial pricing, borough information, and related property attributes. Preprocessing involved removing irrelevant columns (e.g., "Horizontal Enlargement") and handling missing values.

02 — EDA
Exploratory Data Analysis
Price distribution — most apartments cluster around $100K with a smaller group near $500K
Apartment distribution by borough — Staten Island leads, followed by Queens, then Brooklyn
Average initial cost by borough — Queens commands the highest prices near $1M
03 — Insights
Key EDA Findings
  • Queens has the highest initial apartment prices — nearing $1 million on average, making it the premium borough in this dataset.
  • Staten Island leads in apartment count — most apartments listed, suggesting higher supply despite moderate pricing.
  • Brooklyn has the lowest prices — and the least representation in the dataset, suggesting a distinct market profile.
  • Bimodal price distribution — most apartments cluster around $100K, with a smaller group near $500K and virtually nothing in between, suggesting distinct market segments.
04 — Model
Model Building
Algorithm
Ridge Regression with regularization to prevent overfitting on sparse features
Pipeline
One-hot encoding of categorical features + Imputation for missing values
Evaluation Metric
Mean Absolute Error (MAE) — baseline model comparison to measure improvement
Interactivity
ipywidgets added interactive controls for exploratory filtering in Jupyter
05 — Features
Feature Importance
Feature importance — variables most predictive of initial apartment price

Feature importance analysis reveals which property attributes — borough, unit type, building class — carry the most predictive weight. This can guide real estate investors on which variables to prioritize when estimating property value.

06 — Conclusion

This project successfully explored and modeled New York apartment pricing across boroughs. By combining thorough EDA, a clean Ridge regression pipeline, and interactive visualization, the analysis provides both statistical rigor and practical interpretability for real estate pricing decisions.
