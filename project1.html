Clustering Small Business
Owners Using K-Means

Type
Unsupervised ML · Clustering
Techniques
K-Means · PCA · EDA
Libraries
Pandas · NumPy · Matplotlib · Seaborn · Plotly · Scikit-learn
01 — Overview
Objective

This project aimed to analyze financial profiles of small business owners, identify hidden patterns in their financial behavior, and create distinct clusters of owners with similar financial characteristics.

The findings support targeted financial product design, lending decisions, and policy interventions for small business support programs.

↗ View on GitHub
02 — Methodology
Project Pipeline
  • Exploratory Data Analysis (EDA)Analyzed financial profiles of small business owners vs. non-business owners to identify distributional differences and key variables.
  • Data PreprocessingCleaned and standardized financial features using Pandas and NumPy to prepare for modeling.
  • Dimensionality Reduction with PCAApplied Principal Component Analysis (PCA) using Scikit-learn to reduce the feature space and improve cluster separation.
  • K-Means ClusteringSegmented business owners into distinct financial clusters using K-Means. Optimal cluster count determined through elbow method analysis.
  • VisualizationRendered cluster plots and financial distributions using Matplotlib, Seaborn, and Plotly for clear interpretation.
03 — Tools
Technologies Used
Data Manipulation
Pandas · NumPy — data cleaning, transformation, and financial feature engineering
Visualization
Matplotlib · Seaborn · Plotly — cluster plots, distribution charts, and interactive visuals
Machine Learning
Scikit-learn — K-Means clustering and PCA dimensionality reduction
Dimensionality Reduction
PCA — reduces noise and improves the quality and interpretability of clusters
04 — Insights
Key Findings
  • Financial profiles differ significantly between small business owners and non-owners, validating the need for separate segmentation strategies.
  • PCA improved cluster quality by reducing noise from correlated financial features, making clusters more distinct and actionable.
  • K-Means identified distinct financial segments — ranging from high-revenue, asset-rich owners to early-stage, cash-constrained operators.
  • Cluster insights enable targeted interventions — different segments benefit from different financial products, support programs, or lending criteria.
05 — Conclusion

By combining EDA, PCA, and K-Means clustering, this project successfully surfaced meaningful financial segments within the small business owner population. These clusters provide a data-driven foundation for tailored financial services, lending models, and business support policy decisions.
