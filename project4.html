SAS · Excel · Python · Regression

Noiler Chicken
Egg Weight Prediction

Type
Regression · Research
Tools
SAS · Excel · Python (Matplotlib)
Techniques
Multiple Linear Regression · Data Cleaning · Visualization
01 — Overview
Objective

This project aimed to predict the egg weight of Noiler chickens using external egg parameters — specifically egg length, width, and other morphological measurements — through multiple linear regression analysis.

Noiler is a hybrid chicken breed developed for both egg production and meat. Accurately predicting egg weight from external measurements has practical applications in poultry farming, quality grading, and breeding programs.

02 — Tools
Technologies Used
Statistical Analysis
SAS — multiple linear regression modeling, ANOVA, and model diagnostics
Data Management
Microsoft Excel — data cleaning, organization, and initial exploration
Visualization
Python Matplotlib — generating plots for model output and residual analysis
03 — Methodology
Project Pipeline
  • Data Collection & CleaningEgg measurements (length, width, shape index, etc.) collected and cleaned in Excel. Outliers and inconsistencies addressed before modeling.
  • Exploratory AnalysisDescriptive statistics computed to understand the distribution of egg parameters and their relationships with egg weight.
  • Multiple Linear Regression in SASDeveloped regression models to quantify the relationship between external egg parameters and egg weight. Model comparison using R², RMSE, and ANOVA significance tests.
  • Model DiagnosticsResidual analysis performed to validate model assumptions — linearity, homoscedasticity, and normality of errors.
  • Visualization in PythonMatplotlib used to generate regression plots, residual charts, and parameter contribution visuals for reporting.
04 — Insights
Key Findings
  • External parameters are predictive — egg length and width show strong linear relationships with egg weight, validating the regression approach.
  • Multiple predictors improve accuracy — combining several morphological measurements in a multivariate model outperforms any single-variable predictor.
  • Practical value for poultry farming — farmers and processors can use external measurements as a non-destructive proxy for egg weight, enabling faster quality sorting.
  • Consistent patterns across samples — model residuals showed no major violations of regression assumptions, confirming model reliability.
05 — Conclusion

This research project successfully developed multiple linear regression models capable of predicting Noiler chicken egg weight from external morphological parameters. The cross-tool approach — SAS for rigorous statistical modeling, Excel for data management, and Python for visualization — demonstrates strong analytical versatility applied to an agricultural research problem with real-world impact.
